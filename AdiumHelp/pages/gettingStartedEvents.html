Configuring Event Notifications

Adium's event notification system includes support for both audible and visual notifications which can selectively be applied globally, per group, or per contact.

Audible notifications are provided by QuickTime.

Visual notifications are provided by Growl. If Growl isn't installed the first time Adium attempts to post a Growl notification then you will be asked to install Growl; Adium includes a built-in installer for the latest Growl.

The default event preset does not use any Growl notifications. However, Adium comes with a secondary event preset called "Visual Notifications" which will setup a basic set of Growl notices for the most common events. For more in depth information about creating a new event preset please refer to Creating a Personalized Event Preset.

If Growl is not working, please visit our Support forums.
The Growl not doing anything with Adium thread is a good place to start.

For more information on Growl please visit the Growl website.
