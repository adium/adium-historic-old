Creating a Personalized Event Preset

Adium's event notification system includes support for both audible and visual notifications which can selectively be applied globally, per group, or per contact.

To Create a Personalized Event Preset:

  1. Choose Preferences from the Adium menu
  2. Click the Events button
  3. Click the Event preset drop-down menu
  4. Click Add New Preset... from the sub-menu
  5. Enter a title for your new preset and click OK

Now you can begin adding or removing events in order to achieve exactly right set for you. You can also start to explore the many different events that can trigger a notification and the various responses that Adium can supply.

This system of event presets is very powerful for users who need to: occasionally IM in silence but would still like to receive Growl notices, or not be distracted by any notification but still be online; or any other situation you can think of. Now you can create as many presets as you need!

Also, remember that in addition to global events you can also apply per group and per contact events. To setup per group events make sure you have groups visible (choose Show Groups from the View menu) then control click (or right click) on the group you would like to setup up events for and choose Get Info and then click the Events tab. Setting up per contact events works exactly like per group events except you control click a buddy's name and choose Get Info.
