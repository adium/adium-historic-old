Setting Up Accounts

Each instant messaging service has a unique features set which should be considered when creating an account. The pros and cons of each service are beyond the scope of this document and the reader is urged to refer to the official websites.

Registering a New Account
Currently only Bonjour accounts can be created directly from within Adium. To register a new account on all other services please visit their official websites.

Setting Up an Existing Account
If you already have your accounts registered see Setting up an existing account for how to setup them up in Adium.
