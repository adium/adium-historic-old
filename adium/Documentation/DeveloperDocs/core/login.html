AI Login Controller

Manages the Adium user list.

Adium has it's own built in multiple-user support, allowing multiple sets of unique preferences, contact lists, and other information while being logged in under a single OS X user. The multiple-user support is only successful with the cooperation of all preference and file saving code.

Always save user-specific files to the current user directory.


Class Methods

- (NSString *)userDirectory

Returns the directory of the currently logged in user. All user-specific preferences and data files should be stored at this location. This method may return a relative path, such as "~/Library/Application Support/Adium/Users/Adam/"
