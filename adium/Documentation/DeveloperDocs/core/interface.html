AI Interface Controller

Manages Adium's interface

The interface controller handles general interface needs, such as requesting a 'message window' for a handle. The active interface can take any form.

Class Methods

- (NSNotificationCenter *)interfaceNotificationCenter

Returns the shared interface controller notification center. Available notifications:

Interface_ContactSelectionChanged
Sent when the currently selected contact object changes
@"Object" - The newly selected contact object (AIContactObject)
Interface_InitiateMessage
Instructs the interface to open a new 'message window' with the specified contents
@"To" - The destination (AIContactHandle)
@"From" - The source (AIAccount)
@"Content" - The initial outgoing content (This is only guaranteed to be used if a valid 'To' and 'From' are specified)
Interface_CloseMessage
Instructs the interface to close an existing message window
@"Handle" - The handle who's message should be closed
Interface_SendEnteredMessage
Instructs the interface to send the entered message
(object) - Handle who's message to send (AIContactHandle)
Interface_WillSendEnteredMessage
Sent by interface before sending
(object) - Handle who's message is being sent (AIContactHandle)
Interface_DidSendEnteredMessage
Sent by interface after sending
(object) - Handle who's message was sent (AIContactHandle)
